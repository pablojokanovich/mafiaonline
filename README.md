# Mafia Online

Juego de deducción social multijugador en tiempo real.

## Tecnologías
- Frontend: React + Vite + TailwindCSS
- Backend: Node.js + Express + Socket.io
- Base de datos: SQLite (Local) / PostgreSQL (Prod)
- Móvil: Capacitor (Android)

## Instalación

1. Instalar dependencias:
```bash
npm install
cd server && npm install
```

2. Iniciar desarrollo:
- Terminal 1 (Server):
```bash
cd server
npm run dev
```
- Terminal 2 (Cliente):
```bash
npm run dev
```
